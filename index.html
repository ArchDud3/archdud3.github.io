  SkibiOS
SkibiOS is an open-source operating system developed by... me! It is free and open-source, and is hosted on GitHub.
SkibiOS comes in five different versions: Regular, ARM, Pro, Minimal and Lite, being a more complex model than
linux operating sytems but still less complex than the Microsoft, with Pro being priced at $2.50 per twenty-five licenses.
But the main feature of SkibiOS is... the anyapp app, which allows you to use an app image from any operating
system (e.g. .exe file) on SkibiOS! It is still in the concept phase, so a lot of ideas are being created and removed
from this world, but while you are faiting you can check out the official website at skibios.github.io.

This is not a real product (yet) so if someone has copyrighted the name "SkibiOS"
then please let me know through my email at skibioperatingsystem@gmail.com.
Currently SkibiOS has no license, (or code) but it will be licensed with the
GNU General Public license. And no, none of your data will be kept but
